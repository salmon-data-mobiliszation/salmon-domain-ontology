# RDA juvenile-condition case-study fragments

This folder contains the split profile-bridge fragments for the RDA juvenile-condition pilot.

## Current structure

- `08-rda-hakai-measurement-methods.ttl` — Hakai fork-length method-context and measurement terms.
- `08-rda-hakai-method-scheme.ttl` — shared scheme scaffolding for Hakai measurement terms.
- `09-rda-neville-entities.ttl` — entity/stratum-level profile concepts (who/where/what was sampled).
- `09-rda-neville-observations.ttl` — observation and measurement profile concepts (what you measured).
- `09-rda-neville-constraints.ttl` — context, modifiers, and aggregate-statistics vocabulary.

The assembled
`08-rda-case-study-profile-bridges.ttl` and
`09-rda-neville-decomposition-profile-bridges.ttl` files are regenerated from these fragments for downstream ontology imports.
